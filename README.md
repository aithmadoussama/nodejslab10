# 🚀 TP 10 — Application React Complète avec les Hooks

---

## 📌 Description

Ce TP consiste à développer une application React moderne permettant de découvrir et manipuler les principaux Hooks React.

L’application réalisée est une plateforme nommée **Smart Task Hooks** qui contient plusieurs démonstrations interactives des hooks React avec une interface moderne et responsive.

---

## 🎯 Objectifs pédagogiques

✔ Comprendre l’architecture d’une application React  
✔ Utiliser les Hooks fondamentaux  
✔ Gérer l’état local et global  
✔ Manipuler le DOM avec useRef  
✔ Optimiser les performances avec useMemo et useCallback  
✔ Créer des hooks personnalisés  
✔ Mettre en place le routage avec React Router  

---

## 🧰 Technologies utilisées

- ⚛️ React JS  
- 🟨 JavaScript ES6+  
- 🌐 React Router DOM  
- 🧱 HTML5  
- 🎨 CSS3  

---

## 🪝 Hooks utilisés

### 🔹 Hooks fondamentaux
- useState  
- useEffect  
- useContext  
- useReducer  

### 🔹 Hooks avancés
- useMemo  
- useCallback  
- useRef  
- useDebugValue  

### 🔹 Hooks personnalisés
- useLocalStorage  
- useOnlineStatus
